此系统是本人在软中公司实习期间做的一个项目，项目用于帮助企业方便快捷的管理内部业务文件，其实现了文件的上传、下载和在线浏览等功能。

兼容IE8及以上浏览器，Chrome，Firefox，360Chrome等。
